# Тестовое задание БА + СА аналитик

В репозитории находятся материалы по трем задачам тестового задания.

## Состав решения

### Задача 1

BPMN 2.0 модель процесса выдачи ИТ-оборудования пользователю:

- `схема как есть.bpmn` - исходный файл BPMN-модели;
- `схема как есть.svg` - экспорт схемы для быстрого просмотра в браузере или на GitHub.

В модели отражен процесс "как есть", включая вариант с наличием оборудования на складе и вариант с закупкой у поставщика. Также на схеме указаны вопросы и спорные места, которые требуют уточнения для более точного описания процесса.

### Задача 2

Файл `задача 2.md` содержит верхнеуровневую постановку для функциональности публикации товара на маркетплейсе:

- User Story;
- Use Case основного сценария;
- альтернативные сценарии;
- функциональные и нефункциональные требования;
- бизнес-правила;
- диаграмму процесса в Mermaid.

### Задача 3

Файл `задача 3.md` содержит описание REST API регистрации пользователя и алгоритм обработки запроса на стороне бэк-сервиса:

- метод и URL;
- входные параметры;
- успешный ответ;
- ответы с ошибками;
- коды ошибок;
- примеры запроса и ответов;
- пошаговый алгоритм создания пользователя;
- диаграмму алгоритма в Mermaid.

Файл `задача 3-openapi-register.yaml` содержит OpenAPI-спецификацию метода регистрации пользователя.
